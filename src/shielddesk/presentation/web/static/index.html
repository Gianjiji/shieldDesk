Come funziona

  1. Analisi chat — incolla una conversazione (anche export WhatsApp) e ottieni una timeline del rischio, messaggio per messaggio.
  2. Cassaforte — salva come prova solo i messaggi che scegli tu. Restano cifrati a riposo con una catena di hash a prova di manomissione.
  3. Report — esporta un pacchetto PDF+JSON dentro uno ZIP cifrato, con o senza pseudonimizzazione.
